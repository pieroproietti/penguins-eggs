# ChromiumOS family

Calamares installer templates for the ChromiumOS distro family.

Derivatives: ChromiumOS, ChromeOS, FydeOS/openFyde, ThoriumOS, WayneOS, Brunch.

ChromiumOS is Gentoo-derived. Uses dracut for initramfs, Portage for packages,
and /usr/lib64/ for library paths. krill (TUI) is the primary installer since
Calamares availability is limited on ChromiumOS systems.
