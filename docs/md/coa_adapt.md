## coa adapt

Adapt the video resolution to the Virtual Machine window

### Synopsis

The 'adapt' command is a post-boot utility specifically designed for live environments running inside Virtual Machines (such as VirtualBox, VMware, or QEMU/KVM). 

When executed, it forces the guest operating system to dynamically resize its display resolution to perfectly match the current dimensions of the host's VM window, improving the user experience during testing.

```
coa adapt [flags]
```
