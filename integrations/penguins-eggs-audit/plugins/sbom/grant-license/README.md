# grant-license

License compliance scanning for eggs ISO artifacts using [anchore/grant](https://github.com/anchore/grant).
Works with SBOMs generated by [syft-generate](../syft-generate).

## Usage

```bash
# Check licenses during produce (requires --sbom)
eggs produce --sbom --license-check

# Standalone license check against an ISO or SBOM
eggs sbom check <iso-path>
eggs sbom check --policy .grant.yaml <sbom.spdx.json>

# List all licenses in a target
eggs sbom list <iso-path>

# Initialise a default policy file
eggs sbom init-policy
```

## Policy file

```yaml
# .grant.yaml
rules:
  - pattern: "GPL-3.0"
    mode: deny
  - pattern: "MIT"
    mode: allow
  - pattern: "Apache-2.0"
    mode: allow
```

## Dependencies

- `grant` binary on PATH — install from https://github.com/anchore/grant#installation
- `syft` (optional) — for scanning raw ISOs rather than pre-generated SBOMs
